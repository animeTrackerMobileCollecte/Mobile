## Auteurs
- Chadi
- Luderson
- Nassim

- Moncef

## Application mobile AnimeTracker (Expo + React Native) :
- Navigation par onglets (Home, Wishlist, Favorites, Settings)
- Authentification JWT
- Catalogue animé avec recherche/pagination
- Ajout aux listes (wishlist / ongoing / completed)
- Favoris
- Dashboard + DashboardStats avec graphiques

## Stack
- Expo (React Native)
- React Navigation (Stack + Bottom Tabs)
- Axios
- react-native-gifted-charts / chart-kit (selon les écrans)
- AsyncStorage (token)

## Prérequis
- Node.js (LTS recommandé)
- Expo Go installé sur ton téléphone (Android/iOS)
- Un backend démarré (local ou déployé)

## Installation
```bash
npm install
```

## Lancer l'app
```bash
npm start
```
